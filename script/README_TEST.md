# 同花顺API快速测试指南

## 方法一：在同花顺客户端内测试（推荐）

1. **打开同花顺交易客户端**

2. **进入策略交易平台**
   - 点击菜单：工具 → 策略交易平台
   - 或按快捷键：Ctrl + T

3. **创建测试策略**
   - 点击"新建策略"
   - 复制 `script/test_api.py` 的内容
   - 粘贴到编辑器
   - 保存为 "test_api"

4. **运行测试**
   - 点击"运行"按钮
   - 查看输出结果

## 方法二：命令行测试（需要系统Python）

如果你有系统Python环境，可以尝试：

```bash
# 安装依赖
pip install pandas numpy

# 运行测试（可能无法访问同花顺内部API）
python script/test_api.py
```

## 方法三：创建简单测试文件

创建一个简单的测试文件 `script/simple_test.py`：

```python
# 简单测试
from ths_api import *

# 测试行情
api = hq.ths_hq_api()
quote = api.get_quote('600000')
print('浦发银行价格:', quote['600000']['price'])

# 测试交易（只查看，不实际下单）
import xiadan as xd
print('总资产:', xd.g_money['zzc'])
print('持仓数量:', len(xd.g_position))
```

## 测试结果解读

### ✓ 所有测试通过
说明同花顺API工作正常，可以使用完整功能

### ✗ 部分测试失败
- **导入失败**: 检查Python环境和API路径
- **行情失败**: 检查网络连接和交易时间
- **资金/持仓失败**: 可能未登录或账户问题

## 在交易时间测试

建议在交易日 9:30-15:00 之间测试，此时：
- 行情数据实时更新
- 可以测试虚拟下单（模拟账户）

## 安全提示

⚠️ **测试时请注意**：
1. 使用模拟账户测试，避免真实资金风险
2. 不要在生产环境直接运行未测试的代码
3. 测试交易命令时使用 `-notip` 参数外的其他方式
4. 设置好风险控制参数
